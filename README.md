# 铁花入梦｜线上沉浸式打铁花体验

一个以国家级非遗“打铁花”为主题的网页原型。项目通过夜景舞台、匠人击打动作、Canvas 粒子铁花、音效和文化解密内容，模拟“观看表演 - 参与开打 - 理解文化”的线上体验流程。

项目不依赖后端服务，也不使用 Three.js、A-Frame、Blender 或大型前端框架，主要由原生 HTML、CSS 和 JavaScript 实现。

## 项目亮点

- 沉浸式夜景舞台：深色国风视觉、金橙火光、花棚、灯笼、炉火和匠人剪影。
- 打铁花体验页：点击“开打一板”后，匠人先完成蓄势、举杆、前挥、击打、回收动作，再在击打点触发铁花。
- 多种铁花样式：支持“基础喷洒型”和“旋转圆环型”，可在体验页中手动切换。
- 轻量视角交互：电脑端可通过鼠标移动模拟轻微转视角，移动端保留触摸视角逻辑。
- 控制面板折叠：体验页底部控制面板可收起，减少对表演画面的遮挡。
- 文化解密页：包含匠人技艺、高温铁水与危险性、节庆祈福意义、非遗传承现状等内容。
- 纯前端运行：直接打开 `index.html` 或使用 Live Server 即可查看。

## 运行方式

方式一：直接打开

1. 下载或克隆本项目。
2. 在文件夹中双击 `index.html`。
3. 使用 Chrome、Edge 或其他现代浏览器打开。

方式二：使用本地预览服务

1. 在 VS Code 中打开项目文件夹。
2. 使用 Live Server / Live Preview 插件运行 `index.html`。
3. 在浏览器中访问插件提供的本地地址。

说明：浏览器通常会限制自动播放音频。音效需要在用户第一次点击页面按钮后才会播放，这是浏览器策略，不是项目错误。

## 操作说明

- 点击 `进入体验`：进入打铁花表演页。
- 点击 `开打一板`：匠人执行击打动作，并在击打阶段触发铁花。
- 点击 `样式：基础喷洒 / 样式：旋转圆环`：切换铁花粒子样式。
- 点击 `连续演示`：自动循环开打，再次点击可停止。
- 点击 `抬头视角`：快速切到抬头观赏角度。
- 点击 `收起`：折叠底部控制面板。
- 点击 `展开控制`：恢复底部控制面板。
- 移动鼠标：电脑端可轻微上下左右转视角。
- 手机触摸移动：可模拟移动端视角变化。
- 点击 `前往解密页面`：查看打铁花文化内容。

## 技术实现概览

- `index.html`：页面结构、舞台元素、控制按钮、文化解密内容。
- `css/style.css`：视觉风格、响应式布局、匠人剪影、花棚、控制面板、人物击打动画。
- `js/main.js`：页面切换、Canvas 粒子系统、铁花样式、声音控制、视角交互、连续演示和按钮逻辑。

铁花效果使用 Canvas 2D 粒子实现：

- 基础喷洒型：以击打点为起点，模拟铁水向上抛洒并在空中爆散。
- 旋转圆环型：围绕中心生成多层环形粒子，并加入切向速度形成旋涡感。

人物动作使用 CSS keyframes 实现：

- 蓄势
- 举杆
- 前挥
- 击打
- 回收

JavaScript 在击打动作进行到约 `680ms` 时触发现有粒子与音效，使火花与工具落点对齐。

## 资源说明

图片资源位于 `assets/images/`，音频资源位于 `assets/audio/`。

如需替换素材，建议保持原文件名和路径不变。若修改文件名或目录，需要同步更新 `css/style.css` 或 `js/main.js` 中的资源引用路径。

## 项目定位

本项目适合作为设计思维、非遗传播、交互原型或前端视觉实验的课程作品。它并不复现真实打铁花的危险操作，而是通过安全的网页交互帮助用户建立初步兴趣和文化理解。
