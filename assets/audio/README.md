# Audio placeholders

Sound files used by the prototype:

- `ambient.mp3`: low-volume looping night ambience after entering the experience
- `strike.mp3`: metal hit sound for "开打一板"
- `burst.mp3`: spark burst sound
- `whoosh.mp3`: overhead spark fly-by sound
- `ui_click.mp3`: ordinary button click sound

If files are missing, the prototype still runs normally.
